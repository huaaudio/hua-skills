# hua-skills

Skills that I created might be useful for [PicoClaw](https://github.com/sipeed/picoclaw).

## Skills

| Skill | Description |
|-------|-------------|
| [audio-playback](./audio-playback/SKILL.md) | Play audio files (MP3, WAV, OGG, FLAC …) using available CLI tools |
| [audio-record](./audio-record/SKILL.md) | Record audio from a microphone to a file |
| [audio-convert](./audio-convert/SKILL.md) | Convert audio files between formats and adjust encoding settings |

## Installation

Copy the skill folder(s) you want into your PicoClaw skills directory, e.g.:

```bash
cp -r audio-playback audio-record audio-convert ~/.picoclaw/skills/
```
