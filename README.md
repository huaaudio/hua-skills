# hua-skills

Skills that I created might be useful for [PicoClaw](https://github.com/sipeed/picoclaw).

## Skills

| Skill | Description |
|-------|-------------|
| [pico-email](./skills/pico-email/SKILL.md) | Receive and send emails via PicoClaw internal channel |
| [audio-playback](./skills/audio-playback/SKILL.md) | Play audio files (MP3, WAV, OGG, FLAC …) using available CLI tools |
| [audio-record](./skills/audio-record/SKILL.md) | Record audio from a microphone to a file |
| [audio-convert](./skills/audio-convert/SKILL.md) | Convert audio files between formats and adjust encoding settings |

## Installation

Copy the skill folder(s) you want into your PicoClaw skills directory, e.g.:

```bash
cp -r [skills] ~/.picoclaw/skills/
```
